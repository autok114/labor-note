# 노동노트

고용노동부와 대법원 등 공식 출처를 우선하여 노동 이슈, 중요 판례, 짧은 실무 스터디를 정리하는 공개 사이트입니다.

## 자동 업데이트

GitHub Actions가 한국시간 기준 매일 09:17, 13:17, 17:17, 21:17에 실행됩니다.

- 고용노동부 최신 보도자료 확인
- 노동 관련 대법원 판례속보 확인
- 공식 법령·판례 기반 5분 스터디를 한국 날짜 기준 하루 다섯 편 추가
- 공식 원문에서 본문과 게시일 수집
- 주제 분류와 실무 확인사항 생성
- 변경된 데이터 커밋 및 GitHub Pages 자동 배포

수집에 일시적으로 실패하면 기존 데이터를 유지하고 다음 실행 때 다시 확인합니다. 모든 자동 생성 글에는 공식 원문 링크와 출처가 표시됩니다.

첫 적용 시에는 5분 스터디 7편을 한 번에 등록하고, 이후부터는 하루에 다섯 편씩 추가합니다. 준비된 법령·판례 주제를 먼저 사용하고 이후에는 새 공식 자료를 실무 학습형으로 구성합니다. 자동 수집이 하루 네 번 실행되어도 같은 날 스터디가 중복 등록되지는 않습니다.

## 로컬 실행

```bash
npm install
npm run update:data
npm run dev
```

공개용 정적 빌드와 검증은 다음 명령으로 실행합니다.

```bash
npm test
```
